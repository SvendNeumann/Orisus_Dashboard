# Orisus CFO Dashboard

Interner Frontend-Prototyp fuer die Orisus Zahnmedizin MVZ GmbH.

Phase 1 enthaelt ausschliesslich klickbare UI mit realistischen Dummy-Daten:

- Next.js, TypeScript, Tailwind CSS
- shadcn/ui-inspirierte lokale Komponenten
- Recharts Visualisierungen
- simulierter Login-, Erstlogin-, PIN- und Passwort-vergessen-Flow
- CFO Cockpit, Standorte, Standortdetail, Analysen, Plan/Ist, Cashflow, Darlehen & Earn-Out, Uploads und Reports
- keine echte Datenbank, Authentifizierung, Uploadlogik oder externe API-Anbindung

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
